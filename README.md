Tareas
--
* Diseña un sitio web (contenido a elección) de 2 o 3 páginas web que incluya:
* Etiquetas semánticas (html5).
* Enlaces externos e internos.
* Al menos una imágen (html5).
* Al menos una tabla.
* Estilos externos e internos (CSS3).
